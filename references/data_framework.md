# 6层数据框架详细说明

## 框架概述

商务专家skill采用6层数据框架，全面覆盖客户经营的各个方面。每层都有明确的数据来源、处理方式和输出内容。

## Part 1: 客户基础档案分析

### 分析目标
从客户主数据中提取关键信息，生成标准化的客户基础档案。

### 数据来源
- **文件**: `基础数据/客户主数据.xlsx`
- **字段数**: 139个字段
- **关键字段**: 客户简称、真实服务对象、计费ARR、服务阶段、客户状态、所属区域、行业、主要产品、营收规模等

### 处理方式
- **脚本直接处理**：所有字段由Python脚本直接读取和处理
- **标准化输出**：确保每次生成格式一致

```

├── 基础数据/
│   └── 客户主数据.xlsx
├── 订阅合同行/

├── 实施合同行/
│   ├── 固定金额台账.xlsx
│   └── 人天框架台账.xlsx

│   └── 订阅合同收款情况.xlsx
└── 运维工单/
    └── 运维工单.xlsx（可能多个文件）
```

### 文件要求
1. **Excel格式**: .xlsx 或 .xls
2. **编码**: UTF-8 或 GBK
3. **工作表**: 默认使用第一个工作表
4. **列名**: 支持中文列名，脚本通过列索引提取

## 数据处理流程

### 1. 数据加载
```python
# 读取客户主数据
df_basic = pd.read_excel("客户主数据.xlsx")

# 读取订阅数据
df_subscription = pd.read_excel("订阅明细.xlsx")

# 读取实施数据
df_implementation = pd.read_excel("固定金额台账.xlsx")

# 读取运维数据
df_operations = pd.read_excel("运维工单.xlsx")
```

### 2. 数据清洗
- 处理空值和异常值
- 统一日期格式
- 标准化金额单位
- 去重和验证

### 3. 分析生成
- 按模块分别分析
- 生成结构化表格
- 调用LLM进行智能分析
- 整合为完整报告

### 4. 输出保存
- 生成Markdown格式报告
- 保存到指定目录
- 记录处理日志
