```
│   └── 客户主数据.xlsx

```

### 文件要求
1. **Excel格式**: .xlsx 或 .xls
2. **编码**: UTF-8 或 GBK
3. **工作表**: 默认使用第一个工作表
4. **列名**: 支持中文列名，脚本通过列索引提取

## 数据处理流程

### 1. 数据加载
```python
# 读取客户主数据
df_basic = pd.read_excel("客户主数据.xlsx")

# 读取订阅数据

```

### 2. 数据清洗
- 处理空值和异常值
- 统一日期格式
- 标准化金额单位
- 去重和验证

### 3. 分析生成
- 按模块分别分析
- 生成结构化表格
- 调用LLM进行智能分析
- 整合为完整报告

### 4. 输出保存
- 生成Markdown格式报告
- 保存到指定目录
- 记录处理日志
