# 商务专家 - 确定性分析手册

本手册定义客户经营分析的标准化流程和输出格式，确保每次生成结果一致。

## Part 2: 订阅情况分析

### 分析目标
分析客户的订阅信息，包括ARR、订阅周期、产品分布、年度费用变化等。

### 智能分析逻辑
**订阅费用年度变化计算**：
- 读取所有订阅明细行的"订阅有效期从"、"订阅有效期至"、"年订阅费金额"
- 将每个合同按时间阶段拆分
- 叠加同一时间阶段内的所有合同订阅费
- 生成按时间阶段的订阅费用汇总

**示例**：
```
A合同：22年2月 - 27年1月，年订阅25万
B合同：24年12月 - 27年11月，年订阅5万

阶段拆分：
- 22年2月 - 24年11月：25万
- 24年12月 - 27年1月：30万
- 27年2月 - 27年11月：5万
```

### 输出格式

```
## 2. 订阅情况

### 2.1 订阅概览
- 当前ARR：{计费ARR} 元
- 有效订阅总额：{有效订阅费总金额} 元
- 订阅状态：{状态分布}
- 订阅类别：{类别分布}

### 2.2 智能分析
订阅费用年度变化情况（按时间阶段汇总）：
- {阶段1}：{金额}万
- {阶段2}：{金额}万
- {阶段3}：{金额}万
预测：{基于变化趋势的分析}

### 2.3 订阅明细
按年度倒序排列（今年→去年）

{合同名1}
- 产品名称：{产品名}
- 订阅类别：{类别}
- 年订阅费：{金额}
- 有效期：{有效期从} ~ {有效期至}
- 状态：{状态}

{合同名2}
...
```

## Part 3: 实施优化情况分析

### 分析目标
分析客户的实施合同情况，包括合同金额、年度分布、实施优化趋势等。

### 智能分析逻辑
- 固定金额合同：按"合同签订时间"年度分组统计
- 人天框架合同：按"创建日期"年度分组统计

### 输出格式

```
## 3. 实施优化情况

### 3.1 实施概览
按年度体现（2025年→2024年→...）+ 总汇总

#### 2025年
- 固定金额合同：{金额}元
- 人天框架合同：{金额}元
- 小计：{金额}元

#### 2024年
- 固定金额合同：{金额}元
- 人天框架合同：{金额}元
- 小计：{金额}元

#### 总计
- 固定金额合同总额：{固定金额合计} 元
- 人天框架总额：{人天金额合计} 元
- 合同总数：{合同数量}

### 3.2 智能分析
实施优化费用年度变化情况：
- {年份1}年：{金额}万
- {年份2}年：{金额}万
- {年份3}年：{金额}万
预测：{基于变化趋势的分析}

### 3.3 固定金额合同
{合同名}
- 产品名称：{产品名}
- 固定金额：{金额}
- 签订时间：{时间}
- 项目状态：{状态}

### 3.4 人天框架合同
{合同名}
- 产品名称：{产品名}
- 总人天：{人天}
- 应收金额：{应收}
```

### 输出格式

```

### 6.1 客户价值分级
- 价值等级：[A/B/C]级
```

## 内容验证清单

完成分析后，必须验证：

- [ ] Part 1：所有字段已填充，无"无数据"
- [ ] Part 2：订阅费用阶段计算正确，智能分析有依据
- [ ] Part 3：实施费用按年度汇总正确，智能分析有依据
- [ ] Part 4：收款进度计算正确，逾期判定准确
- [ ] Part 5：网络信息已获取（如已执行）
- [ ] Part 6：价值分级有依据，健康度评估有数据支撑
- [ ] 无幻觉：所有数据必须来自原始数据框架，禁止编造
